# InterviewCoach Dataset v1

Русскоязычный синтетический датасет для модуля анализа ответов пользователя с использованием большой языковой модели.

## Назначение

Датасет используется для разработки, проверки и первой итерации дообучения модели, которая оценивает ответы кандидата после сессии технического собеседования.

## Область v1

- специализация: backend
- грейд: junior
- язык: русский
- карточек вопросов: 24
- записей всего: 120
- разбиение: {'eval': 24, 'test': 24, 'train': 72}
- уровни качества ответов: {'good': 24, 'strong': 24, 'empty': 24, 'weak': 24, 'medium': 24}
- темы: {'http_rest': 20, 'api_design': 20, 'jwt_auth': 20, 'sql_indexes': 20, 'transactions': 20, 'caching_redis': 20}

## Состав

- `question_cards/backend_junior/*.json` — карточки вопросов, ключевые пункты, типовые ошибки и подсказки.
- `raw/raw_train.jsonl`, `raw/raw_eval.jsonl`, `raw/raw_test.jsonl` — размеченные ответы для обучения и оценки.
- `export/sft_train.jsonl`, `export/sft_eval.jsonl`, `export/sft_test.jsonl` — формат supervised fine-tuning.
- `coverage_matrix.csv` — покрытие вопросов по темам, качеству ответа и split.
- `annotation_guidelines.md` — правила ручной проверки и доразметки.
- `reports/validation_report.json` — отчёт о структуре датасета.

## Ограничение

Данные являются синтетическими и прошли автоматическую проверку структуры. Перед финальным дообучением 20-30% записей нужно проверить вручную, а спорные оценки скорректировать.
